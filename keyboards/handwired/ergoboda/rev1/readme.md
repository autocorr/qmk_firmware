# Ergoboda Mark 1

## Build the Firmware:

  - Build the firmware with `make handwired/ergoboda/rev1:<keymapname>`
  - This will result in a hex file called `handwired_ergoboda_rev1_<keymapname>.hex`
    `handwired_dactyl_promicro_default.hex`

How to setup your build enviroment can be found here: [Installing Build Tools](https://docs.qmk.fm/#/getting_started_build_tools)
